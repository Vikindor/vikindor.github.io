Privacy Policy

Last updated: April 20, 2026

Scope

This Privacy Policy applies to personal software projects published by Vikindor, including browser extensions, userscripts, websites, web applications, mobile applications, and related project pages, unless a specific product explicitly links to a separate product-specific privacy policy.

The default privacy model for these projects is simple: no ads, no trackers, no embedded analytics, no sale of data, and no collection of personal data unless clearly stated otherwise for a specific product.

What These Projects Do Not Collect

  • No personal data is collected for advertising, profiling, or analytics purposes.
  • No browsing history is sent to the developer.
  • No keystrokes, passwords, payment information, or account credentials are collected.
  • No personal data is sold, rented, or shared with data brokers or advertisers.

Local Processing

Many of these projects are designed to work locally on the user's device or inside the user's browser. If a project reads page content, tabs, URLs, DOM state, browser UI state, local settings, or similar runtime data, that access is used only to provide the product's visible functionality on the user's own device.

Unless a specific product page or listing explicitly states otherwise, this information is not transmitted to the developer or to third-party servers.

Permissions and Access

Some projects may require browser permissions, local storage access, network access to user-requested services, or other technical capabilities that are necessary for their stated functionality. Depending on the product, this can include access to tabs, the active page, page content, context menus, browser storage, or other platform APIs.

Requested permissions or capabilities are intended solely to implement the product's features and are not used for surveillance, advertising, or unrelated data collection.

Local Storage and Settings

Some projects may store preferences locally on the device or in browser-managed storage, for example to remember settings, toggles, filters, or UI preferences.

This data is used only so the product can remember the user's choices. It is not used to identify users, and it is not sent to the developer unless a specific product clearly says so.

Data Transmission and Third-Party Services

By default, these projects do not transmit personal data to the developer's own servers because there are no developer-operated collection endpoints for this purpose.

If a specific project interacts with a third-party service as part of its visible functionality, that interaction will be described in the relevant listing, repository, project page, or product-specific note. If no such disclosure is present, assume the project works locally and does not send your personal data outward.

Advertising, Analytics, and Tracking

  • No advertising SDKs are used.
  • No analytics SDKs or usage telemetry are embedded by default.
  • No cross-site tracking mechanisms are intentionally implemented.
  • No hidden fingerprinting or user profiling is performed.

Data Sharing

No personal data is sold or shared with advertisers, marketing platforms, or unrelated third parties. If a future product requires any exception to this rule, that exception will be disclosed explicitly in the relevant product listing or product-specific privacy notice.

Data Retention

Because these projects are intended to operate locally and not collect personal data for the developer, there is no developer-side retention of personal data.

Local settings stored by a project remain on the user's device or in the user's browser until the user removes them, clears storage, resets the product, or uninstalls the relevant application or extension.

User Controls

  • You can disable or uninstall a browser extension at any time through your browser.
  • You can clear local product storage using browser or platform tools where supported.
  • You can review requested permissions in a browser or extension store listing where applicable.

Children's Privacy

These projects are not designed to knowingly collect personal information from children. If a specific product changes this model, that will need to be disclosed explicitly for that product.

Changes to This Policy

This policy may be updated when browser extension behavior, store requirements, or disclosure needs change. The latest version published at this URL is the current version.
